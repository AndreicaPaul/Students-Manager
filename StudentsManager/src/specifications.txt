Students Manager

The purpose of this app is to give the teacher the possibility of tracking all the students within a classroom.

Features:
1. Add Students - Done
2. View all Students - Done
3. Remove Student - Done
4. Add Disciplines - Done
5. View all Disciplines - Done
6. Remove Discipline - Done
7. Give Student a grade at a certain discipline
8. See all students based on different criteria
    a. name
    b. grades
    c. grades at certain disciplines

UI specs:
The user (teacher) will interact with the app through the console, where a numbered menu will be shown.
